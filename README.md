
**Die Website ist online:** [https://kunst-artlenn.com](https://kunst-artlenn.com)

**Projektbeschreibung:**  
Eine responsive und benutzerfreundliche Website mit klarer Struktur und modernem Design.  



<img width="944" height="466" alt="image" src="https://github.com/user-attachments/assets/be545f50-230f-4a05-9c31-8ab642a7a3aa" />

Die Website dient als **Galerie für einen Künstler** und zeigt seine Werke sowie Informationen über ihn.  
Über die Website ist auch der **Zugang zum Online-Shop** möglich, einfach und bequem zu nutzen. Erstellt mit React, HTML und CSS.

## Technologien
- React
- HTML
- CSS
